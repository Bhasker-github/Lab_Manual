Objective

Learn to create and insert data into a database table.

Steps

  1. Create a Table: Use the following SQL command to create a table in your MySQL database.
    2. 
CREATE TABLE customers (
id INT AUTO_INCREMENT PRIMARY KEY,
name VARCHAR(100),
email VARCHAR(100)
);
  2. Insert Data: Use the following SQL command to insert data into the table.
    3. 
INSERT INTO customers (name, email) VALUES ('John Doe', 'john.doe@example.com');
